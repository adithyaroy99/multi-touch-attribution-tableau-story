# Multi-Touch Attribution Story (Tableau)

Tableau dashboards/story to explore how marketing channels and touchpoint sequences relate to conversion probability.

## What’s inside
- First-touch vs last-touch conversion patterns by channel
- Journey length vs conversion relationship
- Top converting sequences (with minimum sample-size filtering)
- Path performance matrix (volume vs conversion) for prioritisation

## Data
Multi-touch attribution event data from Kaggle (touchpoint-level) + a path-level summary table created in Python for sequence analysis.

## Files
- `.twb` Tableau workbook
- `.pdf` report (method + insights)
- `/assets` exported dashboard images
